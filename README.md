# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.  
This project demonstrates basic front-end development concepts and DOM manipulation.

## Features
- Two-player game (Player X and Player O)
- Interactive and responsive user interface
- Displays game result (win or draw)
- Restart game option

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run the Project
1. Download or clone this repository
2. Open the `index.html` file in any modern web browser
3. Start playing the game

## Project Purpose
This project was created to practice JavaScript logic, event handling, and UI updates using the DOM.  
It is suitable for beginners learning front-end web development.
